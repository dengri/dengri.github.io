Using array syntax in setcookie() function

Array syntax can be used to store key/value pairs in a single cookie. For example:

setcookie( 'arrayName[key]', 'value', time()+3600);

Here is a small example of code, where associative array of parameters is stored in a cookie

<?php

/*
 * Store values in cookie as an array
 * --------------------------------------*/
setcookie('prefs[fontSize]', '25px');
setcookie('prefs[category]', 'News');
setcookie('prefs[screenWidth]', '1024px');

?>

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title></title>
	<style>

		body{
			font-size:<?php echo $_COOKIE['fontSize']; ?>;
		}

	</style>
</head>
<body>
	<p>Hi there! How are You?</p>
	<?php 
		
		if(isset($_COOKIE['prefs'])){

			echo "<ul>";


			/*
			 * Walk through an array and echo values stored in a cookie
			 * ---------------------------------------------------------*/
			foreach($_COOKIE['prefs'] as $key => $val ){

				echo '<li>'. htmlspecialchars($key) .' ---> '. htmlspecialchars($val) .'</li>';
			}


			echo "</ul>";
		}

	?>
</body>
</html>
